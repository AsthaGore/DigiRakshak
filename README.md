# 🛡️ DigiRakshak

**DigiRakshak** is a comprehensive digital safety and cybersecurity awareness platform designed to empower users with the knowledge and skills needed to navigate the digital world safely. Through interactive simulations and educational modules, users learn to identify, prevent, and report various digital threats and scams.

---

## 🚀 Key Features

### 🎮 Interactive Simulators
- **Scam Simulator**: Step-by-step walk-throughs of common digital scams (Lottery, Job, Romance scams, etc.).
- **APK Simulator**: Learn the dangers of sideloading and how to distinguish safe apps from malicious ones.
- **Block & Report Simulator**: Practical training on how to handle and report suspicious contacts on various platforms.

### 🧪 Educational Labs
- **Deepfake Lab**: Understand the technology behind deepfakes and learn the subtle signs used to spot manipulated media.
- **UPI Training**: Mastering safe practices for digital payments and UPI transactions to prevent financial fraud.

### 🧠 Knowledge Assessment
- **Safe or Scam**: A gamified quiz experience where users evaluate real-world scenarios and test their cybersecurity instincts.
- **Simulator Hub**: A central dashboard to access all features and track learning modules.

### 📈 Progress Tracking
- **User Dashboard**: visualize your learning progress, completed modules, and security score improvements.

## 🏗️ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [bun](https://bun.sh/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AsthaGore/DigiRakshak.git
   cd DigiRakshak
   ```

2. Install dependencies:
   ```bash
   npm install
   # OR
   bun install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # OR
   bun dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
